# Hoja Técnica ControlFAC

## Arquitectura
- **Tipo**: SPA (Single Page Application)
- **Framework**: React 18 + Vite
- **Base de Datos**: PostgreSQL
- **Autenticación**: Google OAuth 2.0 / JWT
- **Almacenamiento**: Cloud Object Storage (Imágenes)

## Tecnologías Clave
- React.js
- PostgreSQL
- Tesseract.js (OCR)
- jsPDF (Reportes)
- Lucide Icons
- Google Auth

## Compatibilidad
- **Android**: 8.0+
- **iOS**: 13.0+
- **Cámara**: Requerida (5MP+)
- **Navegadores**: Chrome 90+, Edge 90+, Safari 14+, Firefox 88+
